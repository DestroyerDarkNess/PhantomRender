# Games Tested

This file is a visual gallery of games tested with PhantomRender.

## How To Add A New Game

1. Add your screenshot/image to a folder in the repository (example: `assets/games/`).
2. Add a new section in the gallery below.
3. Include:
   - Game name
   - API used (DX9/DX10/DX11/DX12/OpenGL)
   - Result (`Working`, `Partial`, `Issues`)
   - Optional notes
   - Image link
